# Crypto-Portal
A website to manage your imaginary Crypto wallet.


## Technologies
Java Spring boot and React.

## To run the project

### Spring boot
Run the java application.

### React
In ..\Crypto-Portal\Frontend\crypto-frontend
`npm run dev`
